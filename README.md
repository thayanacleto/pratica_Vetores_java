# Exercícios de Vetores em Java

## Sobre
Repositório com **exercícios de vetores em Java** para praticar declaração, inicialização e manipulação de arrays.  
Inclui iteração com `for` e `foreach`, cálculo de **soma**, **média**, **máximo** e **mínimo**, além de **busca** e **ordenação** de elementos.  

Ideal para **fortalecer lógica de programação** e os fundamentos de arrays em Java.

---

## Recursos
- Prática com **arrays** e manipulação de elementos
- Iteração usando `for` e `foreach`
- Cálculo de **soma**, **média**, **máximo** e **mínimo**
- Implementação de **busca** e **ordenamento**
- Exercícios pensados para **fixar lógica de programação**
